# 📖 Платформа Трезвости | Путь к Свободе

Обучающая платформа для курса по трезвости с премиум-дизайном в стиле старинной книги.

## 🎨 Особенности дизайна

- **Старинная типографика** - использование шрифтов Cormorant Garamond, Cinzel и EB Garamond
- **Премиум-стиль** - изысканные золотые градиенты в лавандовых тонах
- **Архитектура Захи Хадид** - органические формы, многослойность, плавные анимации
- **Эффект перелистывания страниц** - с декоративными орнаментами
- **Фирменный элемент** - голуби как символ легкости и свободы трезвости

## 🚀 Быстрый старт в Replit

### 1. Создание Repl

1. Перейдите на [replit.com](https://replit.com)
2. Нажмите **+ Create Repl**
3. Выберите **Import from GitHub** или **Upload files**
4. Загрузите все файлы проекта

### 2. Автоматическая настройка

После загрузки файлов Replit автоматически:
- ✅ Определит Node.js проект
- ✅ Прочитает `package.json` и установит все зависимости
- ✅ Настроит окружение через `.replit` конфиг
- ✅ Подготовит dev-сервер Vite

### 3. Запуск проекта

Просто нажмите кнопку **▶ Run** или выполните в Shell:

```bash
npm install
npm run dev
```

Проект автоматически откроется в браузере Replit.

- **React 18** - UI библиотека

## 🎨 Дизайн-система

Все цвета и стили определены в `/styles/globals.css` через CSS переменные:
- `--divine-gold`, `--celestial-gold` - золотые градиенты
- `--button-lavender-*` - лавандовые оттенки
- `--sky-*` - небесные тона
- `--success-green` - акцентный зеленый

## 🔧 Решение проблем

### Если проект не запускается:

```bash
# Очистить кэш и переустановить зависимости
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### Если стили не применяются:

- Проверьте, что `/styles/globals.css` корректно импортирован в `main.jsx`
- Убедитесь, что Tailwind v4 настроен в `vite.config.js`
